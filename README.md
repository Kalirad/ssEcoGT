# Stage-structured Ecological Graph Theory (ssEcoGT)

> [!NOTE]
> The methodology presented here is described in depth in Kalirad, A., & Sommer, R. J. (2026). The
> topology of ecological drift. _bioRxiv_, DOI: https://doi.org/10.64898/2026.01.12.698984

# Content of the repository 

+ `ssEcoGT.py`: Stage-structured ecological graph theory (ssEcoGT) model implementation in python.

+ `demo.ipynb`: A notebook that illustrates how to run simulations using ssEcoGT.


# FAQ

### What libraries do I need to run ssEcoGT and generate the results presented in the paper?

The following libraries are required to run the ssEcoGT model:
```ruby
Python   : 3.13.3
networkx : 3.4.2
numpy    : 2.2.5
```
